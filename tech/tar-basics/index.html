[Notes] Tar Basics

發表於 2026-01-08 分類於 tech

一直覺得 tar 的指令很難記。

  • 加入/解開：-c (Create), -x (Extract)
  • 指定檔案： -f (File/Folder)
  • 7Z格式（得到 .tar.gz）：-z
  • 詳細輸出：-v (Verbose)
